# 求一个数n的因数个数（包括n本身和1） 

**基于线性筛框架**
    分解质因数后：n = pow(p1, x) * pow(p2, y);
                  p1, p2都是质数；      x, y是质数的幂；
    方法：    n的因子个数 = (x + 1) * (y + 1);
              例如：72 = pow(2, 3) * pow(3, 2);
              72的因子个数 = (3 + 1) * (2 + 1) = 12个
    条件：f(c) = f(a) * f(b)   c = a * b，且a与b互素(最大公约数 = 1)
          C 的因子个数 = A 的因子个数 * B 的因数个数；
